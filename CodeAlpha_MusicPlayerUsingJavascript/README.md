# 🎵 Melodify — Music Player

A premium, Apple Music-inspired music player built with vanilla HTML, CSS, and JavaScript. Features a clean white UI, animated album art, synchronized lyrics, Web Speech API narration, and a fully functional playback engine using the Web Audio API.

---

## ✨ Features

- 🎨 **Apple Music-inspired UI** — Clean white design with smooth animations and micro-interactions
- 🎧 **10 Built-in Tracks** — Across genres: Electronic, Synthwave, Lo-fi, Jazz, Techno, Ambient, Indie Pop, Hip-Hop, Classical, R&B
- 🔊 **Web Audio API Engine** — Procedurally synthesized audio unique to each track
- 📝 **Synchronized Lyrics** — Lyrics scroll and highlight in time with playback
- 🗣️ **Speak Lyrics** — Web Speech API reads lyrics aloud in real-time
- 🔀 **Shuffle & Repeat** — Full shuffle, repeat-all, and repeat-one modes
- ❤️ **Favourites** — Mark songs as favourite
- 📱 **Fully Responsive** — Works seamlessly on desktop and mobile
- ⌨️ **Keyboard Shortcuts** — Space (play/pause), Arrow keys (seek/volume), S (shuffle), R (repeat), M (mute)
- 🎚️ **Volume Control** — Draggable volume slider with mute toggle
- 🖱️ **Seekable Progress Bar** — Click or drag to seek anywhere in the track

---

## 🎮 Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `→` | Next track |
| `←` | Previous track |
| `↑` | Volume up |
| `↓` | Volume down |
| `M` | Mute / Unmute |
| `S` | Toggle Shuffle |
| `R` | Toggle Repeat |
