# 🧮 Calcify — Premium Calculator

A clean, professional, dark-themed calculator built with vanilla HTML, CSS, and JavaScript. Features a stunning glassmorphism UI with animated background orbs, scientific functions, calculation history, and full keyboard support.

---

## ✨ Features

- 🎨 **Premium Dark UI** — Glassmorphism card, animated glowing orbs, smooth micro-animations
- ➕ **All Arithmetic Operations** — Addition, Subtraction, Multiplication, Division
- 🔬 **Scientific Mode** — sin, cos, tan, log, ln, √, xʸ, π
- 📜 **Calculation History** — Slide-out panel with clickable recall of past results
- ⌨️ **Full Keyboard Support** — Numbers, operators, Enter, Backspace, Escape
- 🔁 **Operator Chaining** — Chain multiple operations without pressing =
- 💡 **Smart Display** — Auto font-sizing for long numbers, expression preview above result
- ⚡ **% and +/− Helpers** — Context-aware percentage and sign toggle
- 📱 **Responsive** — Works seamlessly on mobile and desktop

---

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Number input |
| `+` `-` `*` `/` | Arithmetic operators |
| `Enter` or `=` | Calculate result |
| `Backspace` | Delete last digit |
| `Escape` | Clear all (AC) |
| `%` | Percentage |
| `^` | Power (xʸ) |
